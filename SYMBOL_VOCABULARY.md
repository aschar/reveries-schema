# Symbol vocabulary

16 slugs used for atlas classification and Symbol Card resolution.

| Slug | Name | Jungian register |
|------|------|-----------------|
| water | Water | Unconscious, the maternal, dissolution, renewal |
| shadow | Shadow | The rejected self, the dark double |
| fire | Fire | Transformation, libido, destruction and creation |
| descent | Descent | Katabasis, the underworld journey, depression as initiation |
| ascent | Ascent | Transcendence, inflation, the tower |
| animal | Animal | Instinct, the natural self, the guide |
| house-known | Known house | The self as dwelling, the familiar psyche |
| house-unknown | Unknown house | The unexplored self, the threshold |
| child | Child | The original self, potential, the divine child |
| elder | Elder | The wise old man/woman, accumulated wisdom |
| mother | Mother | The Great Mother, origin, nourishment and devouring |
| father | Father | Authority, law, the senex |
| journey | Journey | Individuation as movement, the quest |
| door | Door | Threshold, transition, the liminal moment |
| eye | Eye | Consciousness, witness, the Self watching |
| fool | Fool | Unenriched, unclassified, the threshold state |
